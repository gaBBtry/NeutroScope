===============================================
   NEUTROSCOPE - CRÉATION EXÉCUTABLE WINDOWS
===============================================

MÉTHODE RAPIDE:
1. Double-cliquez sur "build_windows.bat"
2. Attendez la fin du processus
3. L'exécutable sera dans le dossier "releases/"

PARTAGE SUR ONEDRIVE:
1. Copiez le dossier "releases/" sur votre OneDrive
2. Partagez le lien vers "NeutroScope.exe"
3. Les utilisateurs peuvent lancer directement l'exécutable

PRÉREQUIS:
- Python 3.8+ installé sur Windows
- Connexion internet

PROBLÈMES COURANTS:
- Si erreur "Python non trouvé" : Installez Python depuis python.org
- Si antivirus bloque : Ajoutez une exception pour le dossier "releases/"
- Si build échoue : Vérifiez les messages d'erreur dans la console

DOCUMENTATION COMPLÈTE:
Voir docs/BUILD_WINDOWS.md pour plus de détails
